损失函数（一）

1、Focal Loss 提出的背景

one stage 目标检测算法，存在的问题 ：

1、预测出的bbox 数量非常多， 且 负样本数量 >> 正样本数量 （背景数量 >> 前景数量）

   正/负样本 的定义 ：

  • 正样本： 预测框中包含 object 的为正样本 （预测框与gt bbox 的IoU 大于 IoU threshold）
  • 负样本： 预测框中不包含 object 的为负样本 （预测框与gt bbox 的IoU 小于 IoU threshold）

2、易分的样本数量多，损失主导了训练方向，而难分的样本因为数量少，对 loss 的贡献几乎被忽略

易分类/难分类 的定义 ：

  • 正/负样本 预测为属于 正样本/负样本的概率为 0.95 ===>> 易分类样本
  • 正/负样本 预测为属于 正样本/负样本的概率为 0.25 ===>> 难分类样本/错误分类样本

Focal loss 用于解决上述 问题2 ：易分类样本 和 难分类样本的数量不均衡

问题1 及解决方案作为 baseline ( 原文为 ： an experimental baseline for our proposed focal loss )

3、难分类样本/易分类样本 数量不均衡问题 的解决 : Focal Loss

同 【正负样本数量不均衡问题】解决方案类似：给数量多的易分类样本，较小的损失权重系数；给数量少的难分类样本，较大的权重系数

其中，$(1- p_t)^{\gamma}$ 为调制系数（modulating factor）， 以上公式 3 即为 Focal Loss

  • 对于 $p_t$ ：
    • 当 $p_t \rightarrow 0$ 时，说明正样本预测结果为正的概率很小 （负样本预测结果为负的概率很小），这是一个难分类样本/错误分类样本。这时 $(1-p_t) \rightarrow 1$ ，损失基本不受影响（接近于 标准交叉熵损失值）
    • 当 $p_t \rightarrow 1$ 时，说明正样本预测结果为正的概率很大 （负样本预测结果为负的概率很大），这是一个易分类样本，需要降权，这时 $(1-p_t)$ 是一个较小值，达到降权效果
  • 对于 $\gamma$ ：
    • 当 $\gamma=0$时，Focal loss就是传统的交叉熵损失，我们可以通过调整 $\gamma$ 来实现权重的改变。

4、类别加权 Focal Loss

将两种权重控制一起考虑（既考虑正负样本数量，又考虑难分/易分样本），即为 类别加权 Focal Loss

$\alpha_$ 类别权重

  • $\alpha_t$ 类别权重 -$(1- p_t)^{\gamma}$ 难度权重 论文中证明，在二分类问题中，一般 $\alpha_t = 0.25$， $\gamma = 2$ 时，效果较好
